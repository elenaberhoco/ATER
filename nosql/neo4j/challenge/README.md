# Goodreads Graph Challenge

En janvier 2026, un challenge sur deux jours a été organisé entre l'IUT de Dole et l'IUT de Metz dans le cadre de l'unité d'enseignement "SAE502 : Migration de données vers ou depuis un environnement NoSQL".
J'ai co-construit le sujet du challenge avec l'enseignant en charge de l'UE à Dole.

L'objectif du challenge était de réaliser la migration d'une base de données relationnelle vers une base de données graphe (Neo4j) à partir de données issues du site Goodreads. 

La base relationnelle utilisée pour le challenge a été construite à partir d'un jeu de données extrait de la plateforme en ligne Goodreads, que nous avons nettoyé et restructuré.

Le jeu de données d'origine est disponible en cliquant sur ce [lien](https://cseweb.ucsd.edu/~jmcauley/datasets/goodreads.html#datasets). 
Ce jeu de données est issu des travaux de Mengting Wan et collaborateurs [1,2].

La version nettoyée et restructurée du jeu de données utilisée pour le challenge est disponible sur demande.
